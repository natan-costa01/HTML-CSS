# 📌 Projeto de Site - Alura

Este projeto foi desenvolvido como minha primeira experiência prática utilizando **HTML**, **CSS** e **JavaScript**. O objetivo principal foi aplicar conceitos básicos de desenvolvimento web na criação de um site simples e funcional.

## 🚀 Objetivo do Projeto

Construir um site com múltiplas páginas, aplicando:
- Estruturação com HTML  
- Estilização com CSS  
- Interatividade com JavaScript  
- Boas práticas de organização de código  

---

## 🏠 Página Inicial

A página inicial contém:

- Dois botões com links (âncoras) que direcionam para o site oficial da Alura  
- Estilização personalizada utilizando o arquivo `style.css`  
- Layout baseado em um protótipo desenvolvido no Figma  
- Alterações de cores e organização visual seguindo o design proposto
